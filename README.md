## Tech stack
* MovieDB  API 
* ReactJS
* Material UI
* React-alice-carousel
* redux(soon)

and more


## Installation

Install all the dependencies

```sh
$ npm install
```

## Serve
To serve in the browser  

```sh
$ npm  start
```
